# Duel-of-Sages

A typing racer game.
The game's objective is to test of the fastest sage(typer) between the duellers.

Using:
  - phaser for UI
  - webrtc for P2P communication
  - firestore for webrtc negotiation

Pet project for contact with p2p world.

Still in development.
